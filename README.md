# Software Design Patterns

Individual project created to demonstrate and practice **software design patterns** using Java and Object-Oriented Programming principles.

## Patterns Covered
- Singleton
- Factory Method
- Builder
- Observer
- Strategy
- Adapter

## Technologies
- Java
- Object-Oriented Programming (OOP)

## Project Structure
Each design pattern is implemented in a separate package with clear examples to demonstrate its usage.

## Purpose
This project was developed for educational purposes to improve software design, code reusability, and architectural thinking.
